TARA Atlas

Therapeutic Applications & Risk Assessment

102 BCI techniques mapped to CVSS v4.0 base vectors across 8 categories and 11 hourglass bands. Every attack vector is also a therapeutic application.

What is TARA? Every BCI technique that can harm a patient can also heal one. TARA maps 102 known brain-computer interface techniques across four dimensions:

Modality — Impact severity and physical mechanism
Clinical — Therapeutic applications and FDA status
Diagnostic — DSM-5-TR psychiatric mapping
Governance — Consent tiers and regulations

Switch tabs to view the same grid through each lens. Click any colored cell to see full details.

Recent Additions v1.4 · 2026-02-13

Consumer Device Side-Channel Techniques

3 new techniques extending TARA into consumer device side-channel domains — neural steganography, multi-modal sensor fusion keystroke inference, and display-as-illuminator photometry. WiFi CSI consumer-grade capabilities merged into T0090. All confirmed dual-use with validated clinical applications.

Clinical Applications

  • + ABR audiometry and tinnitus masking therapy (T0100)
  • + Parkinson's detection via keystroke dynamics and tremor characterization (T0101)
  • + Contactless heart rate and SpO2 via screen-based PPG (T0102)
  • + WiFi-based sleep apnea detection, elder care, and COPD monitoring (T0090, expanded)
